# reverseproxy
A reverse proxy for temporary use case.

# Install
Install by script:
```
curl -fsSL https://raw.githubusercontent.com/isayme/go-reverseproxy/refs/heads/master/install.sh | bash
```

Get binary for your os from [Relases](https://github.com/isayme/go-reverseproxy/releases).


# Usage
```
# -t target backend server URL
reverseproxy -t http://127.0.0.1:8888
```
